# mi-primer-html-buenardo
Repositorio de introducción para html dedicado al uso lúdico 

Este es mi proyecto final, en el cual se muestran todos los conocimientos aplicados durante el modulo

Para publicar este sitio en internet utilizando **GitHub Pages**, seguí estos pasos:

1.  **Gestión de Código:** Escribí el código en mi entorno local (VS Code) y utilicé Git para guardar los cambios (commits).
2.  **Repositorio Remoto:** Creé un repositorio público en GitHub y subí mi código local usando `git push`.
3.  **Configuración de Pages:**
    * Fui a la pestaña **Settings** (Configuración) del repositorio.
    * En el menú lateral izquierdo, seleccioné la opción **Pages**.
    * En "Build and deployment", bajo la opción **Branch**, seleccioné `main` y guardé los cambios.
4.  **Verificación:** Esperé unos minutos hasta que GitHub generó la URL pública del sitio y verifiqué que abriera correctamente en el navegador.
